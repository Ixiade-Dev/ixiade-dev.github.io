Le HTML

HTML signifie "HyperText Markup Language". C'est un langage de balises utilisé pour structurer et donner du sens au contenu web. Par exemple : définir des paragraphes, titres et tables de données ou encore intégrer des images ou des vidéos dans une page.

Voici une liste de balises (ou tags) HTML. Pour cette formation, c'est surtout les tags pour la stylisation nous intéresserons.

Note : malgré des balises très diverses, la plupart des pages HTML font face à cette réalité.

Créer un fichier HTML est très simple : ouvrez un éditeur de texte, créez un fichier puis sauvegardez le avec l'extension ".html" !

Voici un petit exemple de HTML :

<div>
    <h3>Un titre</h3>
    <p>Un texte avec des mots en <b>gras</b>, en <i>italiques</i>, <u>soulignés</u>...</p>
    <details>
        <summary>Cliquez ici !</summary>
        <p>Et ce texte apparaît !</p>
    </details>
</div>

Qui donnera ceci :

Un titre

Un texte avec des mots en gras, en italiques, soulignés...

Cliquez ici !

Et ce texte apparaît !

Une page HTML "standard" contient quasiment toujours certains éléments tel que les balises <html> ou <body>. Si vous utilisez Visual Sudio Code, essayez de taper "html" dans un fichier HTML, la fenêtre suivante devrait apparaitre :

Sélectionnez "html:5", voici ce qui devrait apparaitre :

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>

Vous constatez peut être que certains éléments contiennent des informations, ce sont des attributs. Les attributs sont des valeurs supplémentaires qui configurent les éléments ou ajustent leur comportement de différentes manières pour répondre aux critères souhaités par les utilisateurs. Voici un rapide exemple :

<div title="Hello World !">
    Survolez moi !
</div>
Survolez moi !

Le CSS

CSS désigne Cascading Style Sheets (pour Feuilles de style en cascade). C'est un langage de règles de style utilisé pour mettre en forme le contenu HTML. Par exemple : en modifiant la couleur d'arrière-plan ou les polices, ou en disposant le contenu en plusieurs colonnes.

Voyons maintenant comment "écrire" du CSS pour le HTML suivant :

<div>
    Hello World !
</div>
Hello World !

Changeons la couleur de "Hello World" pour du rouge.

<!-- On ouvre une balise "style" pour placer notre CSS -->
<style>
/* On spécifie l'élément sur lequel on applique notre propriété */
div {
    /* On saisit la propriété CSS puis on lui donne sa valeur */
    /* N'oubliez pas le point-virgule pour clore la ligne  */
    color: red;
}
</style>

<div>
    Hello World !
</div>
Hello World !

Maintenant changeons la couleur de l'arrière plan en orange saumon Yoomaneo :

<style>
div {
    color: red;
    background-color: #F4E5CF;
}
</style>

<div>
    Hello World !
</div>
Hello World !

Il y'a plusieurs façon d'implémenter du CSS dans un fichier HTML

Comme nous l'avons déjà vu, avec la balise <style> directement dans le code HTML.
<style>
div {
    text-align: center;
    background-color: red;
}

span {
    color: white;
}
</style>
Ma solution préférée : placer le CSS dans un fichier CSS (extension en .css) puis l'importer dans le fichier HTML
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <!-- On créé l'import dans l'élément <head> du fichier HTML. -->
    <link href="chemin/vers/mon/fichier.css" rel="stylesheet">
</head>
<body>
    
</body>
</html>
Et enfin, il est possible d'ajouter le CSS directement sur un élément avec l'attribut "style"
<div style="color: red; font-weight: bold;">
    Attention à ne pas en abuser ou ça peut vite devenir le bazar !
</div>

Maintenant admettons que nous avons le code HTML suivant :

<div>Je suis vert !</div>
<div>Je suis rouge !</div>

Problème : comment appliquer un CSS différent aux éléments <div> ? La solution : les classes et les ids !

L'attribut HTML "class" permet de créer une référence particulière à un ou plusieurs éléments. Il est possible d'attribuer des propriétés CSS directement à une classe

<style>
/* Pour se référer à une classe dans le CSS, on ajout un "." avant de la nommer */
.green {
    color: green;
}
.red {
    color: red;
}
<div class="green">Je suis vert !</div>
<div class="red">Je suis rouge !</div>
Je suis vert !
Je suis rouge !

Un élément peut avoir plusieurs classes

<div class="green bold purple-background">J'ai plusieurs classes</div>

L'attribut "id" s'apparente à "class", à la différence qu'un certain id ne peut être attribué qu'à un seul élément de la page et qu'un élément HTML ne peut avoir qu'un seul id.

<style>
.green {
    color: green;
}
/* Pour se référer à un id dans le CSS, on ajout un "#" avant de le nommer */
#red {
    color: red;
}
</style>
<div class="green">Je suis vert !</div>
<div id="red">Je suis rouge !</div>
<div class="green">Moi aussi je suis vert !</div>
Je suis vert !
Je suis rouge !
Moi aussi je suis vert !

A noter : les classes et les ids peuvent également être utilisés par JavaScript !

Pour terminer avec le CSS, il est important de noter qu'un style appliqué à un élément se répercutera sur ses "enfants".

<style>
.green {
    color: green;
}
</style>

<div class="green">
    Je suis l'élément parent, et je suis vert !
    <div>
        Je suis l'élément enfant, je suis également vert !
    </div>
</div>
Je suis l'élément parent, et je suis vert !
Je suis l'élément enfant, je suis également vert !
JavaScript

JavaScript est un langage de programmation qui permet de créer du contenu mis à jour de façon dynamique, de contrôler le contenu multimédia, d'animer des images, et tout ce à quoi on peut penser.

JavaScript =/= Java

Enfin, pour la plupart des gens :

Javascript est bien moins simple à appréhender que HTML et CSS et est soumis à des règles de syntaxe plutôt strictes, nous ne l'aborderons pas en profondeur. Voici tout de même un petit exemple :

<div id="my-div">
    <button onclick="changeBackgroundColor()">
        Changer la couleur du fond
    </button>
</div>

<script>
    var myDiv = document.querySelector("#my-div");
    var swapColor = false;
    function changeBackgroundColor() {
        swapColor = !swapColor;
        myDiv.style.backgroundColor = swapColor ? "yellow" : "pink";
    }
</script>
