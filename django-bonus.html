Formation Django - Template de base et Statics

Note : en rouge, variables pouvant être propres à votre programme, pensez à les modifier

Création d'un template de base

Dans votre dossier template, créez un fichier base.html (note : en tapant "html" dans VSCode celui-ci vous proposera des choix. Sélectionner "html:5" vous permet de gagner du temps)

base.html

<!DOCTYPE html>
<html lang="en">
    {% load static %}
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Formation Django</title>
</head>
<header>
    FORMATION DJANGO
</header>
<body>
    {% block content %}
    {% endblock %}
</body>
<footer>---</footer>
</html>

Ensuite, rendez vous sur l'un de vos fichier html


tout en haut
{% extends "*nomdelapp*/base.html" %}
{% block content %}

votre code

tout en bas
{% endblock %}

Tous les templates modifiés de la sorte intégreront le template base.html. Dans le cas présent, cela comprend le head, le header et le footer

Appliquer des statics

Voyons comment ajouter des statics à nos templates. Les statics peuvent être du CSS, du JS, des images, des sons etc.

Nous allons nous focaliser sur l'ajout de CSS. Commençons par créer les bons dossiers dans le répertoire de l'application puis ajoutons-y un fichier "base.css"


    |
    *nomdelapplication
            |
            *static
            |   |
            |   *nomdelapplication
            |           |
            |           *css
            |               |
            |               *base.css
            |
            *templates
            |       |
            |       *nomdelapplication
            |               |
            |               *base.html
            |               *index.html
            *__init__.py
            *admin.py
            *apps.py
            *models.py
            *views.py
            etc.

Ajoutons y un peu de code

base.css

header, footer {
    background-color: aqua;
}

Et enfin, modifier le fichier base.html afin d'ajouter la balise link vers notre fichier css

base.html

    <!DOCTYPE html>
    <html lang="en">
        {% load static %}
    <head>
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <link href="{% static '*nomdelapp*/css/base.css' %}" rel="stylesheet" />
        <title>Formation Django</title>
    </head>
    <header>
        FORMATION DJANGO
    </header>
    <body>
        {% block content %}
        {% endblock %}
    </body>
    <footer>---</footer>
    </html>

Actualisez et vérifiez !
